## ¿Qué es esto?

Un **generador de CV optimizado para ATS** (Applicant Tracking System) completamente gratuito y de código abierto. Todo funciona dentro de tu navegador — no hay base de datos, no hay servidor, no se guarda ningún dato tuyo en ningún lugar.

Cualquier persona en el mundo puede usarlo para crear un CV profesional de calidad y tener una oportunidad real en el mercado laboral.

---

## Características

| Función | Descripción |
|---|---|
| 📝 Formulario completo | Nombre, cargo, contacto, perfil, experiencia, educación, habilidades, idiomas, certificaciones y enlaces |
| 👁 Vista previa en tiempo real | El CV se actualiza mientras escribes |
| 📄 Exportar a PDF | Descarga tu CV directo desde el navegador |
| 🎯 Diseño ATS-friendly | Una columna, sin tablas complejas ni elementos decorativos que confunden a los sistemas ATS |
| 🌙 Modo oscuro / claro | Cambia el tema con un solo clic, guarda tu preferencia |
| 💡 Sugerencias de cargo | Más de 60 títulos profesionales con búsqueda inteligente |
| 🏷 Chips de habilidades | Agrega habilidades rápidamente con Enter |
| 📱 Responsive | Funciona en móvil, tablet y escritorio |
| ✅ Validación de campos | Te avisa si falta información obligatoria antes de exportar |
| 🔒 100% privado | Nada sale de tu navegador. Cero telemetría |

## Cómo usar

1. Abre la página
2. Completa el formulario a la izquierda con tu información
3. Observa la vista previa del CV en tiempo real a la derecha
4. Cuando estés listo, haz clic en **⬇ Exportar PDF**
5. En el diálogo de impresión de tu navegador, elige **"Guardar como PDF"**
6. ¡Listo! Tienes un CV profesional en segundos

> **Consejo:** Para el mejor resultado en PDF, elige tamaño **A4**, márgenes **Ninguno** y activa **"Gráficos de fondo"** en las opciones de impresión.

## Despliegue en GitHub Pages

1. Clona o descarga este repositorio
2. Sube el contenido de la carpeta `Subir/` como raíz de un repositorio en GitHub
3. Ve a **Settings → Pages**
4. En **Source**, selecciona la rama `main` y carpeta `/ (root)`
5. Guarda — en segundos tendrás tu URL pública

```
https://tu-usuario.github.io/nombre-del-repo/
```
